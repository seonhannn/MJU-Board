# 명지대학교 동아리 사이트
> 명지대학교 ICT 대학 소프트웨어 경진대회에 제출한 프로젝트입니다. 

### 인원
	BackEnd - 1명
	FrontEnd - 2명
### Skills
	언어 - Java, JavaScript
	프레임워크 / 라이브러리 - Spring, React, Swiper, React Router

### 구현 목표
	회원가입, 로그인, 메인(동아리 리스트), 동아리 게시판(게시글 목록), 게시글, 작성, 수정
	+ 사용자 인증
